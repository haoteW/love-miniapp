# Contributing

Thanks for contributing to Love MiniApp.

## Ways to contribute
- Report bugs
- Improve documentation
- Submit feature requests
- Open pull requests

## Development
1. Fork repository
2. Create feature branch
3. Test in WeChat DevTools
4. Submit PR with screenshots if UI changed

## Coding standards
- Keep cloud functions independent
- Avoid hardcoded environment IDs
- Never commit secrets or API keys
