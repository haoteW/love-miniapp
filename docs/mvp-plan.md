# MVP 开发计划

## 阶段 1：基础骨架

- 小程序项目配置
- 页面路由
- 全局样式
- `login`、`initUser` 云函数
- 首页恋爱天数展示

## 阶段 2：恋爱日期

- 用户设置恋爱开始日期
- 保存到 `users` 或 `couples`
- 首页读取真实日期并计算天数

## 阶段 3：纪念日

- `anniversaries` 集合 CRUD
- 最近纪念日倒计时
- 提醒字段预留

## 阶段 4：恋爱日记

- `diaries` 集合 CRUD
- 云存储上传图片
- 图片预览与删除

## 阶段 5：约会打卡

- `checkins` 集合 CRUD
- 地点字段
- 图片字段

## 阶段 6：心愿清单

- `wishes` 集合 CRUD
- 标记完成
- 按 `todo/done` 分组

## 阶段 7：双人绑定

- 生成绑定码
- 输入绑定码绑定
- 更新双方 `coupleId`
- 业务数据按 `coupleId` 共享

## 阶段 8：订阅提醒

- 接入订阅消息模板
- 创建 `reminders`
- 定时云函数扫描并发送
